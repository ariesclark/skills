# Changelog

All notable changes to the `fly` plugin are documented here. This project
follows [Semantic Versioning](https://semver.org).

## [0.1.0]

Initial release. The `fly` skill for Fly.io operations: query the org's hosted
Prometheus and VictoriaLogs, run commands on live app machines via SSH, and
connect to the production database. Ships read-only-scoped `allowed-tools` and
reference docs under `skills/fly/references/`.
